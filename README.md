# FinanceFlow - Backend

A single page MERN stack application for personal finance tracking. Built with React, RTK Query, Express js and MongoDB

### Backend packages
- bcryptjs
- cors
- csv-parser
- dotenv
- express
- jsonwebtoken
- mongoose
- multer
- nodemailer
- randomstring
